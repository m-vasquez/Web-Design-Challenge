Summary: Crime and its Potential Factors

In order to try and solve or mitigate crime in the US, we must first understand the basics and set the groundwork for further study, thus, the purpose of this research.

Using the concept of data analytics we can extract useful information from unstructured data. Instead of focusing on causes of crime occurrence we are focusing mainly on potential crime factors using the most current data available from the year 2018. Data visualization tools- Pandas, Matplotlib, and Plotly- were used to aid in exploration and data manipulation.

Inquiry:
  • What influences crime?
  • Is there a correlation between voting trends and crime rates?
  • Can factors, like unemployment, be making an impact on crime?
